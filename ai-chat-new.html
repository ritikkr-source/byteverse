# 🚀 ByteVerse - START HERE ## Quick Start (60 seconds) ### 1. Open the App ``` File → Open → Select: index.html Or drag index.html to browser ``` ### 2. Sign Up Account - Fill in your details - Click "Create Account" - Done! You're in ### 3. Add a Product Go to **Inventory** tab: ``` Product Name: MacBook Pro Stock: 10 Cost: $1200 Selling: $1500 Expiry: 2026-12-31 Click "Add to Inventory" ``` ### 4. Test Features - **Dashboard**: See totals & profit - **Analytics**: View charts & alerts - **Inventory**: Sell products, track stock - **AI Chat**: Ask "How many products?" ### 5. Logout Click "Logout" button (bottom of sidebar) --- ## 📱 Key Pages | Page | What It Does | |------|--------------| | **index.html** | Login/Sign up | | **dashboard.html** | Overview - totals, profit, charts | | **inventory.html** | Add/sell/delete products | | **analytics.html** | Charts, top sellers, alerts | | **ai-chat.html** | Ask AI about inventory | --- ## 🎯 Core Functions ### Add Product ``` Input fields → Fill details → Click "Add to Inventory" ``` ### Sell Product ``` Click "Sell" button in table → Stock decreases by 1 → Profit auto-calculates ``` ### Delete Product ``` Click "Delete" button → Confirm dialog → Product removed ``` ### Ask AI ``` Type question → Click "Ask AI" → AI responds with insights ``` --- ## ⚙️ Setup (If Needed) **Note**: App works without this, but data won't persist. To enable data persistence: 1. Go to: https://supabase.co 2. Create/sign into project 3. Copy credentials to `js/supabase.js` 4. Create tables (see SETUP.md) --- ## 🔧 Troubleshooting | Problem | Fix | |---------|-----| | Page blank | Refresh (Ctrl+F5) | | Can't login | Create account first | | No charts | Clear cache, refresh | | Buttons not working | Check console (F12) | | Data lost | Create Supabase tables | --- ## 📖 Full Documentation - **SETUP.md** - Database configuration - **QUICK_START.md** - Detailed testing guide - **IMPLEMENTATION_SUMMARY.md** - What was built --- ## 💡 Pro Tips ✨ **Add test data quickly:** ``` 1.
